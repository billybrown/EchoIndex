  • borderlist

    A list with a border in between the elements.

  • brandlist

    A list with a special style specific to the brand (usually colored bullets).

  • commalist

    An inline list seperated by a comma.

  • pipelist

    An inline list seperated by a vertical line.

  • nub

    A small visual element to highlight something. Usually a tiny rectangle.

  • Card

    This is a grouping of text and an image that looks like a card. The image goes to the edge of the container and the text is wrapped in a closed box with a background color.

  • Slat

    This is a grouping of text and image(optional) that looks like a slat (a horizontal material sometimes used in architecture/building). It is borderless and the image often is hidden on small screens.

  • ScreenLink

    This is a grouping of text and images where the text overlays on top of the image. The size of the element is determined by the size of the image. The whole thing is a single link

  • Well

    This element is a basic container of content. It has a background and padding between the edge of the elemnt and the inner text. Anything could be inside here including lists, etc.

  • ListBox

    ListBoxes typically hold a list or grid of other CSS objects. They often have a header and a footer.

  • SmallListBox

    Basically the same purpose as ListBox, except this one is very small. Typically appears inside of sidebars, and its' list is typically stacked.

  • Pagination

    Pagination is the menu for navigating a document that has been divided into discrete pages.

  • Breadcrumbs

    Breadcrumbs provide links back to each previous page the user navigated through to get to the current page or—in hierarchical site structures—the parent pages of the current one. Breadcrumbs provide a trail for the user to follow back to the starting or entry point.

  • CaptionImage

    An image with an optional caption and credit. This layout is often used when images are added via a WYSIWYG.

  • HeaderNewsletter

    Newsletter signup form in the header area.

  • HeaderSearch

    Search form in the header area.

  • MobileMenu

    Open and close mobile menu button.

  • PrimaryMenu

    Primary global navigation of the site.

  • SecondaryMenu

    Secondary global navigation of the site.

  • SiteLogo

    Main website logo.

  • HeaderSocial

    Social links in the header area.
